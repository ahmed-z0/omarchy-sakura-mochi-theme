# Sakura Mochi

Sakurazuki's shell with Sakura Mochi's wallpaper and full pink-and-green palette.

## Installation

```bash
omarchy-theme-install https://github.com/ahmed-z0/omarchy-sakura-mochi-theme.git
```

- Wallpaper: `backgrounds/0-sakura-mochi.png`
- Palette: Sakura Mochi's near-black base, bright pink accents, soft cream, and muted green structure
